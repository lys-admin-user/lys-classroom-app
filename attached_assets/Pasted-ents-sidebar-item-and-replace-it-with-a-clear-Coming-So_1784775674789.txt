4. Campus admin signup is a dead end — no path to get linked to a school
When someone picks "School or district administrator" during signup, they're told to sign up as an educator and contact the LYS team. They then land in a full educator dashboard with zero indication of what to do next or how to escalate their account. There is no "Request Admin Access" button, no "Find My School" search, and no in-app way to get elevated.
Recommendations:
Add a post-onboarding "Request Campus Admin Access" card on the educator dashboard that appears when the user initially selected the admin path — it should let them enter their school name/district and submit a request that goes to site_admin for approval (similar to the Team Hub access request pattern that already exists in the platform).
Allow campus admins to be invited by their district admin or site admin via a dedicated invite link that bypasses self-serve onboarding and pre-assigns their org and role.
Add a visible "Your account is pending elevation" status message rather than silently dropping them into an educator dashboard with no context.
